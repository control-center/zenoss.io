Go templates in JSON values

Most values of ServiceDefinition objects may include Go template expressions, with both native and Control Center functions.

The Go template package provides a rich catalog of methods for embedding variables in text. Control Center provides the following, additional functions Go templates.

  • bytesToMB function
    Divide by 1024.
  • child function
    Use the context of a child service.
  • each function
    Iterate over items.
  • parent function
    Use the context of the parent service.
  • percentScale function
    Multiply an item by a percentage.
  • plus function
    Perform integer addition.
