Hosts page

The Hosts page displays information about the individual hosts that are included in all of the resource pools that Control Center is employing.

The Hosts table lists each machine that is assigned to a resource pool.

Two buttons are located immediately above the right side of the table.

  • The Hosts Map button displays a graphical representation of the relative sizes of the hosts in the table, either by the amount of RAM or the number of CPUs in each host.
  • The Host button displays the Add Host dialog.
