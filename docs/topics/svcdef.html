Defining services

  • Service template
    A service template contains a ServiceDefinition object, in JavaScript Object Notation (JSON) format.
  • ServiceDefinition object
    The information that Control Center needs to start and manage a service, in JavaScript Object Notation (JSON) format.
  • Go templates in JSON values
    Most values of ServiceDefinition objects may include Go template expressions, with both native and Control Center functions.
