Web interface reference

The Control Center binary provides an RPC API, which the command-line interface utilizes, and a REST API, which the web interface utilizes.

  • Initial page
    The first time the Control Center interface is started, the Applications page includes only Internal Services in the Applications table.
  • Applications page
    The Applications page displays upon login, and features a table listing both deployed applications and the services that support Control Center.
  • Resource Pools page
    The Resource Pools page lists the resource pools Control Center is employing, with aggregate information about the hosts in each pool.
  • Hosts page
    The Hosts page displays information about the individual hosts that are included in all of the resource pools that Control Center is employing.
  • Logs page
    The Logs page displays the log information that Control Center collects.
