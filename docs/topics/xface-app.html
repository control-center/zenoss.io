Applications page

The Applications page displays upon login, and features a table listing both deployed applications and the services that support Control Center.

Figure 1. Applications page
  • To display the details page of an application, click its name (in this example, Zenoss.core).
  • To display the Internal Services page, click Internal Services.

The space at the bottom-left of the menu bar displays the relative location of the page in the Control Center web interface hierarchy.
