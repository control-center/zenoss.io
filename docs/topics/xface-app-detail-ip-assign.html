IP Assignments table

This table lists the services that have external (public) IP addresses. The services in this table are also listed in the Services table. However, this table highlights IP properties and includes a mechanism (the Assign button) for changing the external IP address, if necessary.

The Assign button displays the Assign IP dialog.

  • Assign IP dialog
    The Assign IP dialog allows you to assign a new external IP address to a service.
